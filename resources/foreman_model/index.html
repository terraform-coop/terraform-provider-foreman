foreman_model¶

Description¶

Vendor-specific hardware model.

Interactivity¶

  • Creatable: YES
  • Deletable: YES
  • Mutable: YES

Example Usage¶

resource "foreman_model" "example" {


  name = "PowerEdge FC630"

}

Argument Reference¶

The following arguments are supported:

  • hardware_model - (Optional; schema.TypeString) Name of the specific hardware model.
  • info - (Optional; schema.TypeString) Additional information about this hardware model.
  • name - (schema.TypeString) The name of the hardware model.
  • vendor_class - (Optional; schema.TypeString) Name or class of the hardware vendor.

Attributes Reference¶

The following attributes are exported:

  • hardware_model - (schema.TypeString) Name of the specific hardware model.
  • info - (schema.TypeString) Additional information about this hardware model.
  • name - (schema.TypeString) The name of the hardware model.
  • vendor_class - (schema.TypeString) Name or class of the hardware vendor.
